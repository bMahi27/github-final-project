# Contribution Guidelines

Welcome Contributors!

Thank you for considering contributing to the github-final-project. This project focuses on calculating simple interest.

## Code Style

Please follow the coding style used in the existing script to maintain consistency.

## Documentation

Ensure your changes are clear and properly documented.

## Testing

Test your changes before submitting to make sure the script works correctly.

## Pull Requests

Submit your contributions through pull requests with a short description of the changes made.

Thank you for your contribution!
